Custom content


    <link href="css/bootstrap.min.css" rel="stylesheet"/>
    <script src="js/coreui-alert.min.js"></script>

    <div id="example-custom-content"></div>

    <script>
        document.addEventListener('DOMContentLoaded', function () {

            // Custom content
            var alertCustom = new CoreUIAlert({
                "type": "alert",
                "view": "danger",
                "title": "Oh snap! You got an error!",
                "message": '<p>Change this and that and try again. Duis mollis, est non commodo luctus, nisi erat porttitor ligula, eget lacinia odio sem nec elit. Cras mattis consectetur purus sit amet fermentum.</p><p><button type="button" class="btn btn-danger">Take this action</button> <button type="button" class="btn btn-secondary">Or do this</button></p>'
            });
            document.querySelector('#example-custom-content').innerHTML = alertCustom.render();
        });
    </script>
                            

    $alert_danger = CoreUI\Alert::danger(
        '<p>Change this and that and try again. Duis mollis, est non commodo luctus, nisi erat porttitor ligula, eget lacinia odio sem nec elit. Cras mattis consectetur purus sit amet fermentum.</p><p><button type="button" class="btn btn-danger">Take this action</button> <button type="button" class="btn btn-secondary">Or do this</button> </p>',
        "Oh snap! You got an error!"
    );

    echo json_encoded($alert_danger); // JSON output danger

Options

Name type default description
view string primary Identify the appearance of the alert. Can have different values: primary, secondary, info, success, warning, danger, light, dark. But there may be other values if you have you will create your personal styles.
title string The title of the alert.
message string The text content of the message. It is allowed html.

Methods

alert.render()

Generates HTML for insertion into the required element.
